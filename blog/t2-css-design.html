Margin, border, and padding are all part of the "box model" that has to do with CSS elements. Margin is the farthest out setting that allows you to get spacing between other elements. Border is the next step in, and is the visible part of the styling. The element that can be changed to pretty much anything that you would want from any border. Padding is the space bewteen the actual object and the border. If you add a 1px padding to something, the border will start 1px out. The padding also shows whatever you set the background as.
