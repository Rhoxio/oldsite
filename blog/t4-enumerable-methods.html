The #map method is a good tool for when you need to add details in to arrays. For example, you can use #map to add a 5 to all of the elements in an array. All you do is call the method on the array and set the block to add it to the variable stated in the block.

The syntax is fairly simple:

array=[1,2,3]
array.map!{|x| x + 5} => [6, 7, 8]

Remember that using #map! and #map are different. #map! returns the value in itself. It will destroy the original array and replace it with the array the block created. Using #map makes it non destructive, so it needs to be set to another variable of your choice.
